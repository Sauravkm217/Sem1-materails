## üéì Exam Strategy

### Time Management:
- **Part A (MCQs):** 30 minutes
- **Part B (Short answers):** 80 minutes (20 min per question)
- **Part C (Long answers):** 80 minutes (40 min per question)
- **Review:** 10 minutes

### Answer Writing Tips:
1. **Define first** - Always start with definition
2. **Use bullet points** - Makes answers clear
3. **Give examples** - Real-world examples score extra marks
4. **Draw diagrams** - For Mind Mapping, Empathy Map, Process flow
5. **Mention importance** - Always conclude with why it matters
